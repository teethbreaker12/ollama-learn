Aplikacja pozwalająca zarządzać budzetem projektu lub organizacji w oparciu o arkusze google

Przycisk "Utwórz wniosek" tworzy wniosek do którego przypisać mozna fakturę w opcji wprowadź fakture. Utworzenie wniosku odbywa sie poprzez wypelnienie formularza zawierajacego nastepsujace informacje:
1. Osoba skladajaca wniosek
2. Deklarowana kwota
3. Źródło (tj. punkt planu budetowego z ktrego ma byc pokryty koszt)

Wnioski zapisywane są wg. przyjętego nazewnictwa tj. "Wniosek nr [numer]/[rok]"


Przycisk "Wprowadź Fakturę" pozwala wprowadzić do arkusza faktrurę, odbywa się to poprzez wypełnienie formularza zawierajacego nastepujace informacje
1. Wystawca faktury
2. Koszt
3. Źródło (tj. punkt planu budetoweg z ktorego ma byc pokryty koszt)
4. Wniosek którego dotyczy fakture

Wgranie pliku PDF. System automatycznie nadaje fakturze numer wg przyjętego nazewnictwa tj "fv[numer]_do_[nr_wniosku]/[rok].pdf"
